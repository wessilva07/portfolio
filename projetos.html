Belotti Tattoo

Meu primeiro projeto foi autoral, no qual utilizei HTML, CSS e JavaScript para criar o site de um amigo que é tatuador. Nesse projeto, empreguei as três tecnologias mencionadas, além do framework de CSS Bootstrap.
No site, você encontrará fotos do estúdio, das tatuagens e um breve resumo sobre como meu amigo começou. Além disso, há uma seção de orçamento na qual utilizei JavaScript para realizar cálculos e também para ativar o botão somente quando todos os campos estiverem preenchidos.

Fala LD

Desenvolvi este projeto com o objetivo de aprimorar minhas habilidades em HTML, CSS, Bootstrap e JavaScript, com um foco especial no aprofundamento do conhecimento em PHP como linguagem back-end. Utilizei o PHP para realizar o envio e recuperação de dados em um banco de dados. O projeto consiste em três páginas, cada uma dedicada a avaliações de diferentes lojas locais. Nestas páginas, os usuários podem fornecer comentários e atribuir uma classificação por estrelas. A plataforma registra e exibe os comentários, juntamente com as avaliações em estrelas, calculando também a média das classificações recebidas. Destaco que o foco principal deste projeto é a interação com o comércio local da minha cidade, onde apresento algumas das lojas em destaque.
Esse foi o projeto que mais me desafiou pois algumas vezes me senti travado pois eu não conseguia trazer para o site os comentarios que ficaram registrados no banco de dados mas no final eu acabei olhando alguns tutoriais e consegui resolver este probelma.
